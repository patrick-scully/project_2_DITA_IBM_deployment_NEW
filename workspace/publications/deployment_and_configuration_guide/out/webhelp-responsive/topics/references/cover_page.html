This table lists the general information about he techniques used for linking and conditional processing and other objectives of the project.

Tasks Details
Develop technical document I have developed a technical document with sections described and included this cover page.
Information Types and elements I have used a range of DITA information types in this project. I also included codeblocks, notes, tables and hazzardstatements where appropriate.
Hierarchial links I have created hierarchial links using the DITA map and added collection-types of "family" within. Each section and its children are linked using a relationship table within the main map.
Cross references I have used xref to all external webpages referenced and used conkeyref to add all images.
Links to DITA and non-DITA resources My linking includes links with DITA (conref, conkeyref, relationship tables) and non-DITA (local PDF and external weblinks) resources. A conref example can be seen on the Solution Components topic using the title Application Stack as an example.
Images All images are referenced using conkeyref.
Glossary I have created a Glossary with the required terms alphabetically. The glossgroup is in alphabetical order itself and also recreated in the map file using topichead and the attribute navtitle.
Index I have created an index withg more than 20 terms. 6 terms have been nested and 5 more have synonymous index terms. The Index has been created using conkeyref. It has been added using metadata to the "About this publication" topic.
